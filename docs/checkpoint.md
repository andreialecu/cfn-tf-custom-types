# Check Point Provider

## Configuration

To configure this resource, you may optionally create an AWS Secrets Manager secret with the name **terraform/checkpoint**. The below arguments may be included as the key/value or JSON properties in the secret:

* `server` - (Optional) Check Point Management server IP.
* `username` - (Optional) Check Point Management admin name.
* `password` - (Optional) Check Point Management admin password.
* `context` - (Optional) Check Point access context - `web_api` or `gaia_api`. Default value is `web_api`.
* `domain` - (Optional) Login to specific domain. Domain can be identified by name or UID.
* `timeout` - (Optional) Timeout in seconds for the Go SDK to complete a transaction. Default value is `10` seconds.
* `port` - (Optional) Port used for connection to the API server. Default value is `443`.
