# Sematext Provider

## Configuration

To configure this resource, you must create an AWS Secrets Manager secret with the name **terraform/sematext**. The below arguments may be included as the key/value or JSON properties in the secret:

* `sematext_region` - (Required) desired Sematext Cloud Region  "US" or "EU";
