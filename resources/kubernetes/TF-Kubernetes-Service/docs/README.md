## Properties

#### WaitForLoadBalancer

Terraform will wait for the load balancer to have at least 1 endpoint before considering the resource created. Defaults to `true`.

_Required_: No

_Type_: Boolean

_Update requires_: [No interruption](https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/using-cfn-updating-stacks-update-behaviors.html#update-no-interrupt)
